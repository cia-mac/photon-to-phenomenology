Which way is it really moving?

what this is
Through a small opening a moving edge shows only motion across itself, so countless true directions look the same. Only the line ends can reveal which way it really moves.
what to do
Drag the bars to set their true motion, then grab the rim and pull to open the aperture.
ⓘ about
Through the slot, every direction looked the same. Your visual system never measured the true motion. It inferred one, and only the ends of the lines could settle the question. Vision is not recording, it is construction.
